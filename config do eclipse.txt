01 - Baixar a versão zipada do JavaFX e extrair para uma pasta.

02 - Baixar o MySQL Connector para Java e salvar em uma pasta.

03 - Instalar o SceneBuilder.

04 - Criar as bibliotecas de usuário antes de criar o projeto:

	# Windows -> Preferences -> Java -> BuildPath -> User Libraries

	# New... -> JavaFX

	# New... -> MySQLConnector

	# JavaFX -> Add External JARs... -> Procurar a pasta do /javafx/libs e selecionar todos os jars

	# MySQLConnector-> Add External JARs... -> Procurar a pasta do /mysqlconector e selecionar o jar.



05 - Indicar os caminhos do SceneBuilder e JavaFX SDK:

	# Windows -> Preferences -> JavaFX
	# SceneBuilder Browse... -> Encontrar o SceneBuilder.exe na pasta de instalação
	# JavaFX 11+ SDK Browse... -> Encontrar a pasta do JavaFX, não há executável.

06 - Criar um novo projeto JavaFX

	# Nome do projeto, Next > Aba Librarie > Modulepath > Add Library... > User Library > Marque as bibliotecas do JavaFX e do MySQLConnector na lista, Next > para mais configs e Finish.
	# Se o arquivo module-info.java indicar erro, mude a primeira linha para module javafx.

07 - Adicione a configuração abaixo.

	# Menu Run -> Run As -> Run Configurations -> Lista Java Application -> Main -> Aba Arguments -> Campo VM Arguments
	# Copiar o conteúdo abaixo, adaptando para sua pasta:
	--module-path C:\java-libs\javafx-sdk\lib --add-modules=javafx.fxml,javafx.controls
	# Close.
